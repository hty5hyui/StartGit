Задание 1: 
1) Создать класс
2) класс должен иметь поле, которое является массивом типа String и в котором записаны имена учащихся в классе (от 5 имен)
3) класс должен иметь поле, которое является массивом типа int и в котором записан средний бал каждого из учеников
4) класс должен имметь метод, который выводит в консоль всех учеников с их средним балом
5) При нажатии на кнопку в главном классе должен выполняться метод из пункта 4



Задание 2: 
1) Создать класс
2) класс должен иметь поле типа String, в котором записано имя учащегося
3) класс должен иметь поле, которое является массивом типа int и в котором записаны оценки ученика (от 5 оценок)
4) класс должен иметь поле типа double, в котором БУДЕТ записан средний бал учащегося
5) класс должен имметь метод, который высчитывает средний бал ученика на основании его оценок и выводит в консоль его имя и средний бал
6) При нажатии на кнопку в главном классе должен выполняться метод из пункта 5
